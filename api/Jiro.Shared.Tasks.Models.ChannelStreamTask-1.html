Struct ChannelStreamTask<TResult>

Namespace
Jiro.Shared.Tasks.Models
Assembly
Jiro.Shared.dll

Represents a pending channel-based stream task that waits for a client to provide a ChannelReader. This struct is used internally by TaskManager to track channel stream requests and their completion sources.
